    • -사이트에 문의한 결과 일시적인 오류 발생이라며 현금결제를 유도하여 현금으로 결제를 마침.
    • -그러나, 이 과정에서 김씨의 카드정보와 주민번호 등이 유출됐고 범인들은 이를 이용해 상품권과 게임머니 등을 구매함.
    • -신용카드 결제 오류가 발생했을 경우 소비자가 사용한 카드사로 연락하여 신용카드 이용내역 확인
    • -추가적인 도용 방지를 위해 신용카드 결제 알림서비스(SMS) 신청 - 기 신청자도 신청여부 확인 필요
    • -피싱사이트로 인한 피해 발생 시 금융감독원(☎1332) 또는 인터넷진흥원(☎118)에 신고
    SNS
    • -트위터 등에서 ‘Did you see this crazy tweet about you?‘ 등 사용자가 호기심을 가질만한 내용으로 위장한 단축 URL을 클릭하면, 트위터 사용자 로그인 페이지와 유사한 웹 사이트로 연결됨.
